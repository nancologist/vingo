# Vingo
Etymology: Video Conference Bingo => Vingo
Author: Morteza Jalilifar

## 1. Run Project
0. ``git clone https://github.com/nancologist/vingo.git``
1. ``cd server/``
2. ``npm install``
3. ``npm start``
4. Open a second terminal.
5. ``cd client/``
6. ``npm install``
7. ``npm start``
